# Lending Club Loan Default Analysis



## Overview



This project analyzes loan-level data from Lending Club to identify

key drivers of default risk and propose risk management recommendations.



## Business Questions



* Is the Lending Club grading system an effective predictor of loan performance?

* Which customer segments have the highest default rates?

* What policies could reduce credit losses?



**Identify variables that significantly correlate with default to optimize the approval process and minimize the risk of default.**



## Dataset



Dataset used: [Lending Club Loan Data](https://www.kaggle.com/datasets/adarshsng/lending-club-loan-data-csv) (DbCL) v1.0 published by [adarshsng](https://www.kaggle.com/adarshsng) on [kaggle.com](https://www.kaggle.com/)

1. Preparation

2. Processing

4. Key Findings

## Key Findings



* The Lending Club grading system is a highly effective predictor of loan performance

* Small business loans and high DTI show elevated risk

* Employment length has minimal impact



## Recommendations



* Tighten underwriting for small business loans

* Introduce DTI caps

* Streamline the application process



## How to Reproduce

1. Download the original dataset and place it in the Data/Raw folder

2. Run scripts 01 and 02 or the .Rmd file
